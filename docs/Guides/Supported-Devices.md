# Passkey Supported Devices

Passkeys are supported on modern phones, tablets and computers with a recent operating system.

| Device Type | Supported From |
| -- | -- |
| iPhone | iOS 16 or later |
| iPad | iPadOS 16 or later |
| Mac | macOS Ventura (13) or later |
| Android Phone | Android 9 or later |
| Windows PC | Windows 10 (1903+) or Windows 11 |
